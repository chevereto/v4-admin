# IP bans

`/dashboard/settings/ip-bans`

IP bans allows to ban IP addresses and IP ranges from using the system.

## Adding IP bans

Click on "Add".

## IP ban values

### Ip address

The IP address to ban.

| Type   | Accepted                |
| ------ | ----------------------- |
| String | IPV4 and IPV6 addresses |

### Expiration date

Determines the expiration date for the IP ban.

| Type     | Optional |
| -------- | -------- |
| Datetime | Y        |

### Description

A brief description about the IP ban (e.g. the reason for the ban).

| Type   | Accepted |
| ------ | -------- |
| String | Any      |
