# Home

`/dashboard`

The **Dashboard Homepage** provides a real-time overview of all activity and resource usage across your instance.

## Chart

The Stats chart displays activity trends over time for the following metrics:

- **Files** — number of files uploaded
- **Disk** — storage space consumed
- **Users** — registered user accounts
- **Albums** — albums created
- **Tags** — tags applied to content

Use the time range buttons to adjust the period shown:

| Button | Period         |
| ------ | -------------- |
| 1W     | Last 7 days    |
| 1M     | Last 30 days   |
| 3M     | Last 3 months  |
| 6M     | Last 6 months  |
| 1Y     | Last 12 months |

The chart toolbar also provides zoom-in, zoom-out, pan, and reset controls for closer inspection of any time range.

## Totals

Below the chart, the dashboard displays the current cumulative totals for the instance:

- **Files** — total number of files stored
- **Albums** — total number of albums created
- **Users** — total number of registered user accounts
- **Tags** — total number of tags in use
- **Disk used** — total storage space consumed by all files
