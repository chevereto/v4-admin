# Tags

`/dashboard/tags`

At **Dashboard Tags** you can browse all the tags that exist in the system. Tags are created automatically when users apply them to files; they cannot be created from this page.

## Listing

Tags are displayed as pill-shaped labels in a flow layout. Each tag is a link to its public tag page at `/tag/{name}`, showing all files tagged with that label.

## Sorting

Use the sort menu next to the **Tags** heading to change the order of the listing.

| Option      | Description                                  |
| ----------- | -------------------------------------------- |
| Most recent | Tags ordered from newest to oldest (default) |
| Oldest      | Tags ordered from oldest to newest           |
| Most viewed | Tags ordered by total view count             |
| A-Z         | Tags ordered alphabetically by name          |

💡 To edit or delete a tag, open it by clicking on it to go to the tag page, where those actions are available.
